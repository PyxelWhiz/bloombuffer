Privacy reminder.

This folder is for the public BloomBuffer app files only.

Do not upload:
- JSON backups with real yard data
- CSV exports with real yard notes
- Photos
- Addresses
- Debug logs
- Contractor, HOA, insurance, or personal notes
- Anything that reveals private home or property details

The public GitHub Pages site should contain app code only.
Your yard inventory should stay in your browser unless you intentionally export and move it.
